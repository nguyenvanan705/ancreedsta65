📊 BIỂU ĐỒ ĐÃ THÊM

- Mục đích: Kết hợp bar chart và line chart

🚨 HỆ THỐNG CẢNH BÁO
====================

Cách hoạt động:
1. Khi vào Dashboard: Hệ thống tự động kiểm tra dữ liệu
2. So sánh với ngưỡng trong cài đặt
3. Hiển thị cảnh báo nếu có vấn đề

Các loại cảnh báo:
- Doanh thu thấp: Khi tổng doanh thu < ngưỡng
- ROI thấp: Khi ROI trung bình < ngưỡng
- Chi nhánh có vấn đề: Khi doanh thu/ROI < 50% ngưỡng
- Tăng trưởng âm: Khi có chi nhánh giảm doanh thu

Cách gửi cảnh báo:
- In-app: Toast notification ngay lập tức
- Email: Gửi email (giả lập)
- Push: Push notification (giả lập)

Tiền tệ (Currency)
- 16+ loại tiền tệ: VND, USD, EUR, GBP, JPY, KRW, CNY
- Khu vực châu Á: Đầy đủ các nước Đông Nam Á

📞 HỖ TRỢ
==========

Lỗi thường gặp
1. Module not found: Chạy npm install hoặc pip install -r requirements.txt
2. Excel read error: Kiểm tra định dạng file Excel
3. Chart not loading: Kiểm tra dữ liệu và API
4. Port 3000 đã được sử dụng: Tắt ứng dụng khác hoặc thay đổi port
5. Dependencies lỗi: Chạy clean.bat để dọn dẹp
6. Node.js không tìm thấy: Cài đặt Node.js từ https://nodejs.org/
7. Build lỗi: Chạy clean.bat để dọn dẹp cache

Performance
- Large datasets: Sử dụng pagination
- Real-time updates: WebSocket cho dữ liệu live
- Caching: Redis cho tối ưu tốc độ
