🚀 Sales Analytics Dashboard
============================

Hệ thống phân tích doanh thu thông minh với AI - Chỉ cần 2 file để quản lý!

📁 Các file quản lý
==================

🚀 start.bat - Khởi động ứng dụng
----------------------------------
Chạy để khởi động development server:
```
start.bat
```

- 🚀 Khởi động ngay lập tức
- 🌐 Server tại http://localhost:3000
- 📊 Sales Analytics Dashboard
- 💻 Nhấn Ctrl+C để dừng

🧹 clean.bat - Dọn dẹp cache
-----------------------------
Chạy khi gặp lỗi hoặc cần reset:
```
clean.bat
```

- 🗑️ Xóa .next directory
- 🗑️ Xóa node_modules/.cache  
- 🗑️ Xóa .turbo directory
- 🗑️ Xóa dist directory
- ✅ Dọn dẹp hoàn tất

🎯 Workflow đơn giản
====================

Lần đầu setup:
1. Cài đặt Node.js từ https://nodejs.org/
2. Chạy `npm install` để cài dependencies
3. Chạy `start.bat` để khởi động

Development hàng ngày:
- Chạy `start.bat` để khởi động

Khi có lỗi:
1. Chạy `clean.bat` để dọn dẹp cache
2. Chạy `start.bat` để khởi động lại

📊 Features chính
================

- 📈 Real-time analytics - Phân tích doanh thu thời gian thực
- 🤖 AI insights - Phân tích và đề xuất từ AI
- 🚨 Alert system - Hệ thống cảnh báo thông minh
- 📊 Interactive charts - Biểu đồ tương tác đẹp mắt
- 📤 Export reports - Xuất báo cáo (PNG, PDF, Excel)
- 🌍 Multi-timezone - Hỗ trợ nhiều múi giờ
- 🌙 Dark/Light theme - Giao diện sáng/tối

🔧 Troubleshooting
=================

Lỗi thường gặp:

1. Port 3000 đã được sử dụng
   - Tắt ứng dụng khác đang chạy trên port 3000
   - Hoặc thay đổi port trong package.json

2. Dependencies lỗi
   - Chạy `clean.bat` để dọn dẹp
   - Chạy `npm install` để cài đặt lại

3. Node.js không tìm thấy
   - Cài đặt Node.js từ https://nodejs.org/
   - Restart terminal sau khi cài đặt

4. Build lỗi
   - Chạy `clean.bat` để dọn dẹp cache
   - Chạy `start.bat` để khởi động lại

📞 Hỗ trợ
=========

Nếu gặp vấn đề, hãy:
1. Kiểm tra console để xem lỗi
2. Chạy `clean.bat` để dọn dẹp
3. Chạy `start.bat` để thử lại
